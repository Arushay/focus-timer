# Focus Timer

A Pomodoro timer application to help you focus in 25-minute sessions.

## Features

- Start, pause, and reset timer
- Add extra time (+1 min, +5 min)
- Tracks total completed sessions
- Saves session history to your browser
- Desktop notifications when time is up

## Technologies Used

- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage API

## How to Use

1. Download or clone this repository
2. Open index.html in any web browser
3. Click Start and begin focusing
